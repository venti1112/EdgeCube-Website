# Server Cores

EdgeCube supports more Minecraft server cores for both Java and Bedrock editions.

## Java Edition

| Core | Description |
|------|-------------|
| **Vanilla** | Official Minecraft vanilla server |
| **Paper** | High-performance server with plugin support (Bukkit/Spigot/Paper API) |
| **Spigot** | Most widely used modified server (CraftBukkit fork) |
| **CraftBukkit** | Original modified server providing the plugin API |
| **Fabric** | Lightweight, modular mod loader |
| **Forge** | Classic mod loader (MinecraftForge) |
| **NeoForge** | Next-generation mod loader |
| **Purpur** | Feature-rich, high-performance Paper fork |
| **Leaf** | Performance-focused Paper fork with new features |
| **Leaves** | Redstone-optimized Paper fork |
| **BungeeCord** | Classic multi-server proxy |
| **Velocity** | Modern high-performance proxy |

## Bedrock Edition

| Core | Description |
|------|-------------|
| **PocketMine-MP** | Classic Bedrock server (PHP) |
| **PowerNukkitX** | High-performance Bedrock server (Nukkit/PNX) |
| **Allay** | High-performance Bedrock server (Java) |

## Creating an Instance

After selecting a core, EdgeCube automatically downloads the server files. You can also:

- **Import existing JAR**: Import server files already on your device
- **Import archive**: Import a zip file for automatic extraction
- **Import modpack**: Import Modrinth `.mrpack` modpacks
