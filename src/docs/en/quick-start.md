# Quick Start

## 1. Download & Install

Get the latest APK from the [Download page](../#download) and install it.

> Requires Android 8.0+. File access permission will be requested on first launch — please allow it.

## 2. Install Runtime

On first use, import a Java runtime:

1. Go to **Manage → Runtime**
2. Tap **Download Runtime** and select a Java version
3. Or import an `.ecpkg` runtime package from your device

## 3. Create a Server

1. On the **Servers** tab, tap **New Instance**
2. Select server type (Java Edition / Bedrock Edition)
3. Choose a server core (Paper / Fabric / Forge etc.)
4. Configure instance name and memory size
5. Tap **Download** or **Import** to add server files

## 4. Start the Server

1. Tap **Start** on the instance card
2. Wait for initialization — status changes to **Running**
3. Open the **Console** to view logs or enter commands

> `server.properties` and other config files are generated automatically on first start.

## 5. Connect to Your Server

- **LAN**: Connect using the local IP on the same network
- **Public**: Enable UPnP port mapping or FRP tunnel to generate a public address to share with friends
