# About EdgeCube

EdgeCube is an open-source, lightweight **Minecraft server management tool for mobile devices**, letting you create, run, and manage Minecraft servers directly on your Android phone.

## Key Features

- **No ROOT Required**: Uses standard Java process management, safe and reliable
- **One-Tap Control**: Start or stop your server with a single tap
- **Real-Time Monitoring**: CPU, memory, and TPS at a glance
- **More Server Cores**: Full support for Paper, Fabric, Forge and more
- **Built-in File Manager**: Full-featured file browser with text editor
- **Remote Access**: FTP / SFTP / SSH all included
- **Port Mapping**: UPnP + FRP tunnel for easy multiplayer
