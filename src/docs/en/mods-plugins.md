# Mod & Plugin Management

EdgeCube features powerful mod and plugin management with support for Modrinth and Poggit.

## Mod Management (Fabric / Forge / NeoForge)

### Online Search

Built-in **Modrinth** search lets you filter by name, loader, and game version. One-click download and install.

### Batch Management

- Enable / disable mods
- Check for updates, batch update with one click
- View dependency relationships
- Import mod files from your device

### Version Selection

Choose mod versions during download. Preview version details (game version, loader, SHA1).

## Plugin Management (Bukkit / Spigot / Paper)

### Poggit Plugins

Search and download PocketMine-MP plugins from **Poggit**, sorted by downloads or last updated.

### Modrinth Plugins

Bukkit-based servers (Paper etc.) can use the Modrinth plugin source.

## File Operations

- Import `.jar` / `.phar` files from your device
- Download queue management with real-time progress
- Automatic detection of `plugins` and `mods` directories
