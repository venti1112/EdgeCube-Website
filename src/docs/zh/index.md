# 关于 EdgeCube

EdgeCube 是一款开源、轻量的 **移动端 Minecraft 服务端管理工具**，让你在 Android 手机上直接创建、运行和管理 Minecraft 服务器。

## 核心特性

- **无需 ROOT**：使用标准 Java 进程管理，安全可靠
- **一键启停**：轻点即可启动或停止服务端
- **实时监控**：CPU、内存一目了然
- **多种服务端核心**：Paper、Fabric、Forge 等主流核心全覆盖
- **内置文件管理**：全功能文件浏览器 + 文本编辑器
- **远程访问**：FTP / SFTP / SSH 一应俱全
- **内网穿透**：UPnP + FRP 隧道，轻松联机
