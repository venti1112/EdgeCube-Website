# 远程访问

EdgeCube 内置三种远程访问方式，方便你在电脑上管理服务端。

## FTP 文件管理

在同一局域网内通过 FTP 客户端访问实例目录。

- 支持匿名访问和账号密码认证
- 可控制读写权限
- 支持 IPv6
- 切换实例或修改配置时自动重启

### 配置

1. 进入 **管理 → FTP 文件管理**
2. 设置端口、用户名和密码
3. 开启服务后用 FTP 客户端连接显示的地址

## SFTP / SSH

通过 SSH 协议提供安全的文件传输和终端访问。

- **SFTP**：通过 sftp 客户端安全传输文件
- **SSH 终端**：通过 ssh 客户端进入设备交互式终端
- 共用同一端口和账号
- 账号密码登录，不支持匿名
- 读写权限分离控制

## MCP 服务

以 Streamable HTTP 协议提供 MCP 服务，供 AI Agent 连接。

- **只读模式**：AI 仅能读取状态、日志和在线玩家信息
- **控制模式**：AI 可启动/停止服务端、发送命令、切换实例
- **Shell 模式**：允许 AI 执行 Shell 命令（高风险）
- Bearer Token 鉴权
