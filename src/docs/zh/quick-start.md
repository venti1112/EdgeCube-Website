# 快速开始

## 1. 下载安装

从 [下载页](../#download) 获取最新版本 APK 并安装。

> 需要 Android 7.0+，首次打开时会请求文件访问权限，请允许以正常使用。

## 2. 安装运行环境

首次使用时需导入 Java 运行环境：

1. 进入 **管理 → 运行环境**
2. 点击 **下载运行环境**，选择合适的 Java 版本
3. 或从外部导入 `.ecpkg` 运行环境包

## 3. 创建服务端

1. 在 **服务器** 页点击 **新建实例**
2. 选择服务端类型（Java 版 / 基岩版）
3. 选择服务端核心（Paper / Fabric / Forge 等）
4. 配置实例名称和内存大小
5. 点击 **下载** 或 **导入** 服务端文件

## 4. 启动服务端

1. 在实例卡片上点击 **启动**
2. 等待初始化完成，状态变为 **运行中**
3. 进入 **控制台** 查看日志或输入指令

> 首次启动会自动生成 `server.properties` 等配置文件。

## 5. 连接服务器

- **局域网**：使用同网络下的设备直接连接内网地址
- **公网**：启用 UPnP 端口映射或 FRP 隧道，生成公网地址分享给好友
