# 模组与插件管理

EdgeCube 内置强大的模组和插件管理功能，支持 Modrinth 和 Poggit 两大平台。

## 模组管理 (Fabric / Forge / NeoForge)

### 在线搜索

内置 **Modrinth** 搜索，可按名称、加载器、游戏版本筛选，一键下载安装。

### 批量管理

- 启用 / 禁用模组
- 检查更新，一键批量更新
- 查看依赖关系
- 支持从设备导入模组文件

### 版本选择

下载时可选择模组版本，支持预览版本详情（游戏版本、加载器、SHA1）。

## 插件管理 (Bukkit / Spigot / Paper)

### Poggit 插件

支持从 **Poggit** 搜索和下载 PocketMine-MP 插件，可按下载量、更新时间排序。

### Modrinth 插件

Paper 等 Bukkit 系服务端可使用 Modrinth 插件源。

## 文件操作

- 支持从设备导入 `.jar` / `.phar` 文件
- 下载队列管理，实时显示进度
- 自动识别 plugins 和 mods 目录
