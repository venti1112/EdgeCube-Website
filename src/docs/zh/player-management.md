# 玩家管理

EdgeCube 提供完整的玩家管理功能，涵盖在线玩家、白名单、封禁和 OP 管理。

## 在线玩家

服务端运行后自动显示在线玩家列表。

- 查看在线玩家名称
- 手动刷新（发送 `list` 指令）
- 踢出玩家（可选原因）

## 白名单

控制谁可以加入服务器。

- 添加 / 移除白名单玩家
- 配合 `enforce-whitelist` 可强制白名单

## 封禁管理

### 玩家封禁

- 封禁 / 解封玩家
- 可选封禁原因和到期时间
- 查看封禁列表

### IP 封禁

- 封禁 / 解封 IP 地址
- 查看 IP 封禁列表

## OP 管理

- 给予 / 取消 OP 权限
- 查看 OP 列表

## server.properties 编辑器

图形化编辑服务器配置文件，无需记忆配置项名称：

- **基础设置**：MOTD、端口、最大玩家数、难度
- **游戏玩法**：PvP、飞行、命令方块、游戏模式
- **网络设置**：正版验证、IPv6、Query、RCON
- **性能设置**：视距、模拟距离、网络压缩
- **资源包**：URL、SHA1、强制使用
